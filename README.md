# 📄 Resume Screening using NLP & Machine Learning

## 🚀 Project Overview

This project is an AI-powered Resume Screening System that classifies resumes into job categories using Natural Language Processing (NLP).

## 🌐 Web Application

Built using Streamlit, the app allows users to:

* Paste or upload resume text
* Predict job category instantly
* Visualize dataset insights

## 🛠️ Tech Stack

* Python
* Scikit-learn
* NLP (TF-IDF)
* Streamlit
* Pandas, Numpy

## 📊 Features

* Resume classification
* Real-time prediction
* Category distribution graph
* Word cloud visualization

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🎯 Output

* Accurate classification of resumes into roles like Data Science, HR, Web Developer, etc.
